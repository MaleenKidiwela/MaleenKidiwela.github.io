About Me

MK
Maleen Kidiwela
Seismologist • Geophysicist • Data Scientist

I'm a seismologist specializing in ambient noise seismology and ocean-bottom geophysics. My research focuses on extracting meaningful Earth structure and dynamic signals from the continuous hum of seismic noise — listening to what the planet tells us when no earthquakes are shaking it.

I work extensively with Ocean Observatories Initiative (OOI) data, monitoring submarine volcanic activity at Axial Seamount and tracking seismic velocity changes along the Cascadia subduction zone. When I'm not wrangling terabytes of waveform data, I'm building tools to automate and visualize complex geophysical datasets.

Python ObsPy NoisePy Jupyter MATLAB GMT SeisComp Linux / HPC

Selected projects in seismology & geophysical data science

Axial Seamount Monitoring

Automated fetch and visualization of OOI hydrophone & seismometer data from Axial Seamount — an active submarine volcano on the Juan de Fuca Ridge. Tracks seismicity patterns and eruptive signals in near real-time.

Python ObsPy OOI API

Cascadia DVV Analysis

Seismic velocity change (dv/v) analysis along the Cascadia subduction zone using ambient noise cross-correlation. Probes stress accumulation and slow-slip events in one of the world's most seismically active margins.

Python NoisePy SeisNoise.jl
GitHub
🌊

Ambient Noise OOI

Processing and analysis of continuous ambient seismic noise from Ocean Observatories Initiative ocean-bottom seismometers. Extracts empirical Green's functions and monitors subsurface structure through noise interferometry.

Python ObsPy HPC
